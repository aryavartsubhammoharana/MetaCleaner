About MetaCleaner

Every picture you take with your smartphone, every document you draft in an office suite, and every voice memo you record contains hidden identifiers known as metadata. While useful for organization, this metadata poses severe privacy risks.

What is Metadata?

Metadata is "data about data." It is embedded directly inside the file structure and is invisible during standard viewing. For example:

  • Photos: Contain GPS coordinates, camera model, lens aperture, creation date, and device serial numbers.
  • Documents: Contain author names, revision history, corporate template names, and local file paths.
  • Media Files: Contain comments, encoder signatures, and artist handles.

Why Clean Metadata?

If you upload a photo to social media or send an invoice to a client, they can extract the file metadata and discover where you live, what software you use, and which organization built the document. MetaCleaner strips all this away, leaving only the raw file contents.

Technical Design

MetaCleaner uses a modular backend built on Python Flask, combined with highly focused data processors:

  • Pillow & pi-heif: Safely load image pixels and write them into clean JPEG, PNG, WebP, and HEIC containers, dropping all tags.
  • pypdf: Rebuilds PDF pages without standard document metadata streams.
  • ZipFile XML Parser: Unpacks open-XML office documents, blanking metadata entries in docProps/core.xml and rebuilds the packages.
  • Mutagen & FFmpeg: Decodes and clears audio/video tags cleanly.
